# 17 · SEO & Open Graph

## Meta
- Title (≤ 60 chars), Description (≤ 155 chars), canonical URLs.

## OG Images
- Dynamic `/og?o=78` etc., 1200×630, PNG/JPEG.

## Sitemaps & Robots
- `/sitemap.xml`, `/robots.txt` allow public pages.

## Internal Linking
- CTA from result to checkout; related resources.
